# 4-Bit Binary Counter using D Flip-Flops

## Overview

This folder contains a 4-bit binary counter designed using KiCad. The circuit is implemented using 74LS74 D Flip-Flops and demonstrates the operation of a binary counter using sequential logic.

## Software Used

- KiCad

## Circuit Description

The design consists of cascaded 74LS74 D Flip-Flops driven by a pulse clock source to generate a 4-bit binary counting sequence.

The schematic and PCB layout were created using KiCad.

## Learning Outcome

- Understanding binary counter operation
- Understanding D Flip-Flop based sequential circuits
- Learning schematic capture using KiCad
- Learning basic PCB layout design
